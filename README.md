# FitTrack - Fitness Tracker Website

A modern, responsive website for a fitness tracker app. This project showcases a complete landing page with features, pricing, and user engagement sections.

## Features

✨ **Modern Design**
- Clean and professional UI with gradient backgrounds
- Responsive design that works on all devices
- Smooth animations and transitions

📱 **Sections Included**
- **Navigation Bar** - Sticky navigation with smooth scrolling
- **Hero Section** - Eye-catching banner with app preview mockup
- **Features** - 6 key features showcasing the app capabilities
- **Pricing** - 3 pricing tiers with feature lists
- **Download** - Call-to-action buttons for all platforms
- **Newsletter** - Email subscription form
- **Contact** - Contact form and business information
- **Footer** - Links, social media, and legal information

🎨 **Design Features**
- Gradient backgrounds and modern color scheme
- Card-based layout with hover effects
- Intersection Observer for scroll animations
- Mobile-first responsive design
- Professional typography and spacing

⚡ **Interactive Elements**
- Smooth scroll navigation
- Form validation and submission handling
- Toast notifications for user feedback
- Active navigation highlighting
- Button press animations
- Parallax effects on hero section
